13.4. Issue Courtesy Refund
Submit a Courtesy Refund Request. Use this function when it is simply to issue the customer a courtesy refund that does
not require RMA shipment returned to seller. This function is a request and will need user to use Get Request Status (listed
in the next section) to receive update about the status of the Courtesy Refund creation. Please note that commission will
not be refunded for refunds issued as courtesy refund.
13.4.1. Resource URL & Information
Resource URL
https://api.newegg.com/marketplace/servicemgmt/courtesyrefund/new?sellerid={sellerid}
Note: Please make sure your request URL is all in lower case (except for Seller ID) and cannot contain any blank spaces or line breaks.

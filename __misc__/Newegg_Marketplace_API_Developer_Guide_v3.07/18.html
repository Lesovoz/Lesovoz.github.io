7.3. Bad Request
Newegg Marketplace API will return a bad request (HTTP status code = 400) if you submitted an inappropriate
request in the following types:

Illegal request body. Such as not well-formatted XML or incorrect tag or data type.
Example: XML, Response
<?xml version="1.0" encoding="utf-8"?>
<Errors xmlns:xsd="http://www.w3.org/2001/XMLSchema"
xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
<Error>
<Code>CE003</Code>
<Message>XML parsing error. The 'NeweggAPIRequest1' start tag on line 1 position 2 does not match
the end tag of 'NeweggAPIRequest'. Line 1, position 451.</Message>
</Error>
</Errors>

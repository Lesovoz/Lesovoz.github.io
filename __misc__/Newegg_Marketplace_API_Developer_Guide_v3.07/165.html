Dimensional Weight
If the dimensional weight is greater than actual weight, FedEx uses the package's dimensional weight as the billable weight.
Example:
A package measuring 26" x 16" x 16" has a volume of 6,656 cubic inches. The dimensional weight equals 6,656
divided by 166 or 41 lbs. If the package's actual weight is 24 lbs, the dimensional weight may be used for the billable weight.
-
Billable Weight
For large packages, FedEx determines the billable weight by taking the package dimensions into account. For large
packages, if the dimensional weight is greater than the actual weight, FedEx may use the dimensional weight instead of the
actual weight.

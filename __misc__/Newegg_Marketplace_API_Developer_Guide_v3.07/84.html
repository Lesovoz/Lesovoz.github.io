9.9. International Program - Get Item Inventory
You can use this API call to track the inventory quantity of items in all registered warehouses around the world, including
the United States.
9.9.1. Resource URL & Information
Resource URL
https://api.newegg.com/marketplace/contentmgmt/item/international/inventory?sellerid={sellerid}
Note: Please make sure your request URL is all in lower case (except for Seller ID) and cannot contain any blank spaces or line breaks.

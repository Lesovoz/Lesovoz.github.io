CheckoutMAP
Integer
0: False
1: True
If Checkout MAP is marked as True, then the
website will ask customer to add item to
shopping cart AND go to checkout page to see
the item's price. The MAP value will not be
considered. Another words, if this field is
marked as true, it doesn't matter if the selling
price is above or below MAP, the customer will
always need to add item to cart and go to
checkout to see item's price.
If MAP is greater than 0.00 and CheckoutMAP is
left blank, by default CheckoutMAP will equal
False.

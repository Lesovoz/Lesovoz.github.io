10.3. Order Confirmation
Use this function to let Newegg mark the orders you have confirmed and downloaded. You will be able to exclude these
downloaded orders in the Get Order Info API’s through OrderDownloaded filter.
10.3.1. Resource URL & Information
Resource URL
https://api.newegg.com/marketplace/ordermgmt/orderstatus/orders/confirmation?sellerid={sellerid}
Note: Please make sure your request URL is all in lower case (except for Seller ID) and cannot contain any blank spaces or line breaks.
